# Katas_Python
# 🐍 Proyecto Katas Python

**Autor:** Alejandro Villanueva.

**Descripción:** Resolución de 41 katas en Python enfocadas en lógica de programación, estructuras de datos, programación funcional y programación orientada a objetos.

---

## 📌 Objetivo del proyecto

El objetivo de este proyecto es aplicar y demostrar los conocimientos adquiridos en Python mediante la resolución de distintos ejercicios (katas), cubriendo:

* Tipos de datos básicos
* Estructuras de datos (listas, diccionarios, tuplas)
* Condicionales y bucles
* Funciones
* Programación funcional (`map`, `filter`, `reduce`, `lambda`)
* Manejo de excepciones
* Programación orientada a objetos (OOP)

---

## 🧠 Metodología

Para la realización del proyecto se ha seguido el siguiente enfoque:

1. **Análisis del enunciado** de cada kata para entender el problema.
2. **Diseño de la solución** priorizando claridad y simplicidad.
3. **Implementación mediante funciones** independientes para cada ejercicio.
4. Uso de:
   * `map()` para transformación de datos
   * `filter()` para filtrado
   * `reduce()` para agregación
5. **Manejo de errores** mediante `try/except` y excepciones personalizadas.
6. Aplicación de **Programación Orientada a Objetos** en las katas:
   * Clase `Arbol`
   * Clase `UsuarioBanco`
7. Mejora de la **legibilidad en terminal** mediante una función auxiliar:
   ```python
   titulo_kata(numero)
   ```
   
---

## ⚙️ Estructura del código

El proyecto se ha desarrollado en un único archivo `.py`, donde:

* Cada kata está separada mediante comentarios claros.
* Cada ejercicio se implementa como una función independiente.
* Se incluye un bloque de prueba tras cada kata para verificar su funcionamiento.
* Se ha priorizado la claridad y organización del código.

---

## 🧪 Ejecución

Para ejecutar el proyecto:

python Katas_Python.py

El programa mostrará en terminal los resultados de cada kata, diferenciados mediante títulos para facilitar su lectura.
